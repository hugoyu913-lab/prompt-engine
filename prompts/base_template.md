# Base Prompt Template

Create a hyper-realistic social media photograph of:

`{subject}`

Style the image with the `{aesthetic}` aesthetic in `{location}`. The mood should feel `{mood}`. Use `{camera}` photography characteristics and compose for `{platform}`.

Core quality direction:

- hyper-realistic photography
- believable human details
- realistic skin texture
- natural facial asymmetry
- cinematic lighting
- authentic social media composition
- realistic lens behavior
- natural shadows and highlights
- non-AI-looking image
- editorial-level taste without over-polishing

Final prompt should read as one clean image-generation prompt.
